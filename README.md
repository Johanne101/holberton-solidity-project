# holberton-solidity-project
Holberton School Project

## Setup:

### Install

```
npm i
```

### Add .config.json and sign up for alchemy key https://alchemyapi.io/

```
{
    "alchemyKey": ""
}
```

### Run tests

```
npm run test
```
