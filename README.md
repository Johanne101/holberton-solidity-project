# holberton-solidity-project

### Assignment

Build a simple NFT exchange. See full writeup in `/assignment` folder.

### Setup:

1. Fork the repository

2. Install

```
npm i
```

3. Add `.config.json` and sign up for alchemy key https://alchemyapi.io/

```
{
    "alchemyKey": ""
}
```

4. Run tests

```
npm run test
```
